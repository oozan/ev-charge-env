# EVChargeEnv — Green Agent Submission

This repository contains an EV charging simulation environment for the AgentX-AgentBeats Green Agent Challenge.

## Structure

- `env/ev_charge_env.py` — Gym environment
- `agent/baseline_agent.py` — Baseline random agent
- `run_evaluation.py` — Entry point executed by AgentBeats
- `Dockerfile` — For reproducible container
- `sample_output.json` — Example output
